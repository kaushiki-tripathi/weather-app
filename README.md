# 🌤️ Weather App

This is a simple and responsive weather web app that allows users to:

- 🌍 Search current weather conditions by **typing a city name**
- 📡 Fetch weather data using **OpenWeatherMap API**
- 📸 See relevant weather icons based on conditions (e.g., rain, snow, clear)
- ⚠️ Display error messages if the city is not found

---

## 🚀 Features

- Real-time weather updates
- Shows temperature, wind speed, and humidity
- Responsive card-style layout
- Custom weather icons based on conditions
- Error message for invalid city names

---

## 🔧 Technologies Used

- HTML
- CSS
- JavaScript
- OpenWeatherMap API
